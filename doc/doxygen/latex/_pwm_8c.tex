\subsection{Detailed Description}
Pulse width modulation module driver. {\bfseries Project} 1814 I\-O\-Lab Remote Sensor Application

{\bfseries Description} The pulse width modulation module contains functions definitions and macros that set up and control the M\-S\-P430 timer B peripheral used to generate pulse width modulated signals.
