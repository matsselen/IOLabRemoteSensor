\subsection{Detailed Description}
Clock configuration module. {\bfseries Project} 1814 I\-O\-Lab Remote Sensor Application

{\bfseries Description} This file provides functions to configure the Unified Clock System (U\-C\-S) on M\-S\-P430x5xx microcontrollers. It also handles any crystal failure interrupts.
