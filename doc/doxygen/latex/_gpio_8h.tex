\subsection{Detailed Description}
G\-P\-I\-O configuration and control macros and functions. {\bfseries Project} 1814 I\-O\-Lab Remote Sensor Application

{\bfseries Description} The functions and macros defined in this file provide access to the configuration and control registers for the G\-P\-I\-O pins on the M\-S\-P430. This includes access to the interrupt control registers for ports 1 and 2.
