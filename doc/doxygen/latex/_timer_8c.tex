\subsection{Detailed Description}
Hardware timer module. {\bfseries Project} 1814 I\-O\-Lab Remote Sensor Application

{\bfseries Description} Provides functions to setup the hardware timers. The control of the timers and their interrupts are handled using the macros defined in the header file.
