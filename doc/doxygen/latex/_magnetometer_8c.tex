\subsection{Detailed Description}
Magnetometer sensor interface module. {\bfseries Project} 1814 I\-O\-Lab Remote Sensor Application

{\bfseries Description} This module provides logic to interface with the magnetometer (3-\/axis Hall effect sensor). This logic assumes that the magnetometer in use is the Freescale M\-A\-G3110 I2\-C Digital Magnetometer.
