\subsection{Detailed Description}
The system tick. {\bfseries Project} 1814 I\-O\-Lab Remote Sensor Application

{\bfseries Description} The timer tick will provide a system tick for triggering periodic events.\-This module provides an interface to register an I\-S\-R that will be called every \char`\"{}tick\char`\"{} or to request a timeout after a given number of ticks.
