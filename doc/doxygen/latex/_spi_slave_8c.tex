\subsection{Detailed Description}
S\-P\-I Slave Module. {\bfseries Project} 1814 I\-O\-Lab Remote Sensor Application

{\bfseries Description} This module is used when this device is a S\-P\-I slave device. The functions and macros defined in the module can be used to communicate with a S\-P\-I master device.
