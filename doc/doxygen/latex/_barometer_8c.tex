\subsection{Detailed Description}
Driver for the barometer sensor. {\bfseries Project} 1814 I\-O\-Lab Remote Sensor Application

{\bfseries Description} This module interfaces to a S\-P\-I barometer sensor to read the barometric pressure and temperature data. This driver is designed to work with a Freescale M\-P\-L115\-A1 S\-P\-I Barometer.
