uint8_t  address,
I2cOperation_t  operation 
)

Transmit the Start Bit on the I2C port, immediately followed by sending the selected Slave Address out to the I2C port.

This assumes that the I2C port has been configured as a MASTER device. Slave devices do not send out the Start bit.

Parameters
[in]port- Selected I2C port.
[in]address- Slave address to communicate with.
[in]operation- Select I2C Write or I2C Read.
Returns
True if the Start Bit and address were sent OK. Otherwise, false.
