Detailed Description

Driver for the gyroscope sensor.

Project 1814 IOLab Remote Sensor Application

Description This module interfaces to a SPI gyroscope sensor to read the 3-axis data. This driver is designed to work with a ST Micro L3GD20 SPI Gyroscope.
