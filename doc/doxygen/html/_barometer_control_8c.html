BarometerSampleArray_t _lastGoodSample = { .data = 0u }
static

Keeps a copy of the last valid barometer reading. Occassionally the barometer will return bad data (all zeros or FFs) that is likely caused by a glitch on the barometer because we are reading it too fast or the SPI clock rate is too high. Regardless of the reason, buffer the last value and report the last good value when reading samples if the new sample is bad.
