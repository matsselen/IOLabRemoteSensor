Detailed Description

Contains the definitions of the Key-Value pairs.

Project 1814 IOLab Remote Sensor Application

Description Contains the definitions of the Key-Value pairs used for configuring the different sensors in the IOLab system. The keys and values defined in this module are based on the document 1814F08 Data Protocol Key-Value table.
