void Adc_Init ( void  )

This function initializes the ADC variables.

Each channel is sampled for about 3 microseconds. Each conversion (sample time + conversion time) takes about 6 microseconds.

Note
The reference voltage takes 75 microseconds to settle (typical) and the temperature sensor, which is sourced by the reference voltage, requires a minimum sampling time of 30 microseconds.
