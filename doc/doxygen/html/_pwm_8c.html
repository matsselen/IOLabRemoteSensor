Detailed Description

Pulse width modulation module driver.

Project 1814 IOLab Remote Sensor Application

Description The pulse width modulation module contains functions definitions and macros that set up and control the MSP430 timer B peripheral used to generate pulse width modulated signals.

#define PWM_CLK   (1200000ul)

The PWM clock is the PWM timer source divided by the timer divider. the SMCLK divided by 20. The timer source is defined in the init to be SMCLK, which is 24MHz. The divider is defined in the init to be 20. Change this define if the init is changed.
