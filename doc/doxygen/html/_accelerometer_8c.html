Detailed Description

Accelerometer sensor interface module.

Project 1814 IOLab Remote Sensor Application

Description This module provides logic to interface with the 3-axis accelerometer. This logic assumes that the accelerometer in use is the Freescale MMA8451Q I2C digital accelerometer.
