void Uart_WriteString ( UartPort_t  port,
const char_t  string[] 
)

Transmit a null-terminated string via the UART. The string is loaded into the transmit buffer to be transmitted using the UART Tx interrupt.

Only the number of characters that will fit into the buffer will be loaded. All other characters in the string will be discarded. This means that the string may not be sent if the buffer is full.

Parameters
[in]port- Selected UART port.
[in]string- String to transmit
