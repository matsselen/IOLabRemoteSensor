Detailed Description

The hardware driver for the quadrature encoder.

Project 1814 IOLab Remote Sensor Application

Description The quadrature encoder uses a a direction and count input to count how many pulses and in which direction a specially designed encoder moved.

The encoder module uses timer A2 of for hardware counting and direction checking.
