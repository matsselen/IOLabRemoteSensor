Function Documentation

void Clock_Init ( void  )

Setup the system clocks. An external 24 MHz oscillator is used to source ACLK and SMCLK as the reference clock source.

ACLK is sourced by XT2 divided by 32 at 750 KHz. SMCLK is sourced by XT2 divided by 1 at 24 MHz. MCLK is sourced by the XT2 divided by 1 at 24 MHz.
