Detailed Description

Magnetometer sensor interface module.

Project 1814 IOLab Remote Sensor Application

Description This module provides logic to interface with the magnetometer (3-axis Hall effect sensor). This logic assumes that the magnetometer in use is the Freescale MAG3110 I2C Digital Magnetometer.
