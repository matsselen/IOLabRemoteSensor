void Dma_Arm ( DmaChannel_t  channel,
void *  sourceAddress,
void *  destinationAddress,
uint16_t  dmaTransferSize 
)

Arms the DMA to make a DMA transaction as soon as it is enabled and triggered.

Parameters
[in]channel- the DMA channel to arm
[in]sourceAddress- the address that is the source of the DMA transfer
[in]destinationAddress- the address that is the destination of the DMA transfer
[in]dmaTransferSize- the number of DMA transfers to make in a transaction
Note
This function does not enable the DMA. Use Dma_Enable or the enable macro to enable the DMA.
