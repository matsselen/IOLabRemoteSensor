Detailed Description

Driver for the barometer sensor.

Project 1814 IOLab Remote Sensor Application

Description This module interfaces to a SPI barometer sensor to read the barometric pressure and temperature data. This driver is designed to work with a Freescale MPL115A1 SPI Barometer.
