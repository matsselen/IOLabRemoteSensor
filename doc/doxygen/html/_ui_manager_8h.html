PlaySequenceStatus_t UiManager_PlaySequence ( UiSequence_t  sequence,
PlayMode_t  mode 
)

Plays a preset sequence that the UI manager can indicate an event to the user. The UI Manager can only play one sequence at a time, so it will not play another sequence if one is currently in progress. It does post an event to the event queue when the sequence is finished playing so that another sequence can be played if needed.

Parameters
[in]sequence- sequence to play to the user
[in]mode- the play mode to play the sequence with.
Returns
The status if the sequence was played and the reason if it wasn't.
