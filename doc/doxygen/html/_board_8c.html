Function Documentation

void Board_Init ( void  )

Initializes the board to a good default state. This includes configuring the GPIO pins, system clocks, power management module (PMM), system interrupts, and the watchdog timer.

This should be one of the first Init functions called so that the board (in particular, the GPIO pins) are put in a good state ASAP.
