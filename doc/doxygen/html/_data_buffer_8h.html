Detailed Description

Data buffer is the memory space reserved to store sensor data.

Project 1814 IOLab Remote Sensor Application

Description The data buffer contains memory space to store sensor data and can be written to and read from. The buffer contains locking mechanisms to ensure that data is not compromised and is valid.
